## 📱 How to Work from Another Device

### Option 1: Using GitHub Desktop (Easiest)

1. **Download GitHub Desktop** from [desktop.github.com](https://desktop.github.com)

2. **Sign in** with your GitHub account

3. **Clone the repository:**
   - Click "Clone a repository from the Internet"
   - Select "Damian-delevoper/Dyo-Web"
   - Click "Clone"
   - Choose where to save it (e.g., Documents)

4. **Start working!** Edit files, make changes

5. **Save and Push:**
   - Type a commit message (e.g., "Updated product descriptions")
   - Click "Commit to main"
   - Click "Push origin"

6. **Sync from other device:**
   - Click "Fetch origin" to download latest changes
   - If changes detected, click "Pull origin"

---

### Option 2: Using Command Line (More Control)

#### First Time Setup on New Device:

```bash
# 1. Install Git (if not already installed)
# Windows: Download from git-scm.com
# Mac: brew install git
# Linux: sudo apt install git

# 2. Clone your repository
git clone https://github.com/Damian-delevoper/Dyo-Web.git

# 3. Enter the folder
cd Dyo-Web

# 4. You're ready to work!
```

#### Daily Work from Any Device:

```bash
# STEP 1: Get latest changes from GitHub
git pull

# STEP 2: Make your changes (edit files)

# STEP 3: Save your changes
git add .
git commit -m "Description of what you changed"

# STEP 4: Upload to GitHub (syncs all devices)
git push
```

## 🔄 Sync Between Devices

### Desktop → Laptop:

**On Desktop:**
```bash

git commit -m "Updated content"
git push
```

**On Laptop:**
```bash

```

### Laptop → Desktop:

**On Laptop:**
